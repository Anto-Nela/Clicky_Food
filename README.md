# Clicky_Food

As you can see, my naming skills are subpar xD. This is a 2D game made for and by following the Unity Junior Programmer Pathway.
In this game, you must make sure to click the food to destroy it before it leaves the screen, otherwise it's game over. Be careful of clicking on anything other than food. 

Made in 3D environment and Unity version 6000.3.8f1.
